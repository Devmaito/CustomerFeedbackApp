# Customer Feedback App

A simple web application for collecting customer feedback and saving data to Google Sheets.

## Features
- Name, email, message form
- Sends feedback to JSONPlaceholder (Test API)
- Stores data in Google Sheets
- Displays submitted feedback on the page

## How to Use
1. Open `index.html`
2. Fill the form
3. Submit to see your feedback and store it in Google Sheets
